Table Reservation System:

A table reservation system built using HTML, CSS and JavaScript.
Technologies used:
HTML, CSS and JS
Type of project:
Solo, relevant to bar manager job.
Status:
The project is still in progress, but the aim is to create a system where users can select a date and time, and the system will check if there are any available tables at that time. If there are, the user can then make a reservation. The system will also have an admin page where the restaurant can view and manage all reservations.
